# sql_data_analysis
# SQL Data Analysis Practice

This repository contains basic SQL queries used for practicing data analysis and database operations.

## Skills Demonstrated
- SELECT queries
- Filtering using WHERE
- Sorting using ORDER BY
- Aggregation functions (COUNT, AVG, SUM)
- GROUP BY operations
- JOIN operations

## Database Scenario
The dataset represents a small **e-commerce system** containing:
- Customers
- Products
- Orders

## Tools Used
- MySQL / PostgreSQL
- SQL

## Purpose
To practice SQL queries used for retrieving and analyzing business data, which is commonly required in troubleshooting and data investigation roles.
